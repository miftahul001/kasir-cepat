# ⚡ Kasir Cepat

A super fast, keyboard-first Point of Sale (POS) system built for small shops (warung) and UMKM.

Designed to be:
- ⚡ Ultra fast (no lag, no waiting)
- ⌨️ Keyboard-driven (no mouse needed)
- 📴 Offline-first (works without internet)
- 🪶 Lightweight (vanilla JavaScript, no framework)

---

## 🚀 Features

- Instant product search (fuzzy search)
- Keyboard navigation (↑ ↓ + Enter)
- Smart suggestion box
- Cart management (add, remove, qty control)
- IndexedDB storage (offline transactions)
- Real-time total & change calculation

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Enter (search) | Add product |
| ↑ ↓ | Navigate suggestion / cart |
| + | Increase quantity |
| - | Decrease quantity |
| Delete | Remove item |
| F2 | Focus search |
| Enter (global) | Checkout |

---

## 🧱 Tech Stack

- Vanilla JavaScript
- IndexedDB (offline storage)
- Vite (development & build)
